# Dial Tracker

A lightweight tool to log dials, connects, and bookings throughout the day.

- Tap **Dial** / **Connect** / **Booked** to log each one as it happens
- Tap any number to manually correct it
- Shows today's totals plus a 7-day breakdown with connect rate and booking rate
- Each person's numbers are stored locally in their own browser — nothing is shared or synced between users

No login, no backend. Just open the link and start tapping.
